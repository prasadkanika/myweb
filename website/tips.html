Tips

  • Reading inputs which are in same line

    • input: 1 2 3 4
    Syntax: list1=[int(k) for k in input().split()] or
    Syntax: list2=list(map(int,input().split()))

    NOTE: list is a keyword so don't use it as a variable
  • printing elements in a list in a same line

    • list1=[1,2,3,4]
    output: 1 2 3 4
    Syntax: print(*list1,end=' ') or
    Syntax: print(' '.join(list1))

    NOTE:
    • '*' unpack the list and return each element
    • ''.join() takes a list and join all the elements the resul is a type string. joinig the list based on what you are giving in the ''.join
  • all() function

    • The python all() function accepts an iterable object (such as list, dictionary, etc.). It returns true if all items in passed iterable are true. Otherwise, it returns False. If the iterable object is empty, the all() function returns True.
    l=[True,False,1]
    print(all(l))    #False
    l=[1<=9,5==5,97>8]
    print(all(l))    #True

  • any() function

    • Python any() function accepts iterable (list, tuple, dictionary etc.) as an argument and return true if any of the element in iterable is true, else it returns false. If iterable is empty then any() method returns false
    l=[True,False,1]
    print(any(l))    #True
    l=[1<=9,5==5,97>8]
    print(any(l))    #True
